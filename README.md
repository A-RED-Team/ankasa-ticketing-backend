## API Documentation

### Auth Endpoint

| No  | HTTP Method | URL                             | Operation                           |
| --- | ----------- | ------------------------------- | ----------------------------------- |
| 1   | POST        | /api/auth/register              | Register new user                   |
| 2   | POST        | /api/auth/login                 | Login user                          |
| 3   | GET         | /api/auth/verify-email          | Verify email user                   |


### User Endpoint
